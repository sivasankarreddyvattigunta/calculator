# calculator

```markdown
## Description
This Python calculator program allows users to perform basic mathematical operations, including addition, subtraction, multiplication, division, modulo calculation, and exponentiation.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with error handling for division by zero)
- Modulo calculation (with error handling for division by zero)
- Exponentiation

## Installation
1. Clone this repository to your local machine:
   ```
   git clone https://github.com/sivasankarreddyvattigunta/calculator.git
   ```

2. Navigate to the project directory:
   ```
   cd calculator
   ```

3. Run the calculator:
   ```
   python calculator.py
   ```

## Usage
- When prompted, enter the desired operation (e.g., 'add', 'subtract').
- Input the first and second numbers.
- View the calculated result.

## Supported Operations
- `add`: Addition
- `subtract`: Subtraction
- `multiply`: Multiplication
- `divide`: Division
- `modulo`: Modulo calculation
- `power`: Exponentiation
```
